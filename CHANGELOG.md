# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [1.0.1] - 2025-01-XX

### 新增
- 交互式模式（`-i, --interactive`），通过问答方式配置参数
- JSON 输出模式（`-j, --json`），方便程序化处理
- 实时进度反馈，显示分析进度
- 友好的错误提示和解决建议

### 改进
- 优化了分析性能
- 改进了报告的可读性
- 增强了错误处理机制

### 修复
- 修复了某些边缘情况下的数据统计问题

## [1.0.0] - 2025-01-XX

### 新增
- 核心能力分析：贡献度、影响力、协作度等指标
- 独自维护指标：分析在项目中的独立性
- 代码精炼度：重构中的代码优化情况
- 技术广度：涉及的技术栈和模块数量
- 情感分析：通过提交消息分析编码心境
- 时间分析：工作习惯、黄金时段、极端时刻
- 六维雷达图：活跃度、影响力、精炼度、协作度、稳定性、广度
- 支持指定年份、采样文件数量等参数
- 自动检测 Git 用户信息
